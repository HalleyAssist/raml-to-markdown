"# ISO Data Service (isodata) API documentation version 1.0.0\n\n---\n\n## Countries\nWork with `Country` objects for all countries in the system. All `name` properties get automatically translated by the service depending on a request's `accept-language` header.\n\n### /countries\n\n#### **GET**:\nList of `Country` objects index by the id of a country.\n\n###### Headers\n\n| Name | Type | Description | Required | Examples |\n|:-----|:----:|:------------|:--------:|---------:|\n| Accept-Language | string | Header containing a sorted list of accepted languages. | false | ``` de, en-gb;q=0.8, en;q=0.7 ``` \n\n### Response code: 200\n\n#### CountryMap (application/json) \n\n```\n{\n  \"DE\": {\n      \"id\": \"DE\",\n      \"alpha3Code\": \"DEU\",\n      \"number\": 276,\n      \"name\": \"Euro\",\n      \"countryCode\": \"49\"\n  },\n  \"US\": {\n      \"id\": \"US\",\n      \"alpha3Code\": \"USA\",\n      \"number\": 840,\n      \"name\": \"United States of America\",\n      \"countryCode\": \"1\"\n  }\n}\n ```\n\n##### *CountryMap*:\n| Name | Type | Description | Required | Pattern |\n|:-----|:----:|:------------|:--------:|--------:|\n| // | object | Object representing a single country item. | true |  |\n\n| Name | Type | Description | Required | Pattern |\n|:-----|:----:|:------------|:--------:|--------:|\n| id | string | Two character country codes. | true | \\w{2} |\n| alpha3Code | string | Alpha-3-Code | true | \\w{3} |\n| number | integer | Number | true |  |\n| name | string | Translated name of a country. | true |  |\n| countryCode | string | Internation phone dial country code. | true |  |\n\n---\n\n### /countries/{id}\n\n* **id**: Identifier of a country.\n    * Type: string\n    * Pattern: \\w{3}\n    * Required: true\n\n#### **GET**:\nSingle `Country` object.\n\n###### Headers\n\n| Name | Type | Description | Required | Examples |\n|:-----|:----:|:------------|:--------:|---------:|\n| Accept-Language | string | Header containing a sorted list of accepted languages. | false | ``` de, en-gb;q=0.8, en;q=0.7 ``` \n\n### Response code: 200\n\n#### Country (application/json) \nObject representing a single country item.\n\n```\n{\n    \"id\": \"DE\",\n    \"alpha3Code\": \"DEU\",\n    \"number\": 276,\n    \"name\": \"Euro\",\n    \"countryCode\": \"49\"\n}\n ```\n\n##### *Country*:\n| Name | Type | Description | Required | Pattern |\n|:-----|:----:|:------------|:--------:|--------:|\n| id | string | Two character country codes. | true | \\w{2} |\n| alpha3Code | string | Alpha-3-Code | true | \\w{3} |\n| number | integer | Number | true |  |\n| name | string | Translated name of a country. | true |  |\n| countryCode | string | Internation phone dial country code. | true |  |\n\n### Response code: 404\nSent if no object for the passed `id` was found.\n\n#### Error (application/json) \n\n```\n{\n    \"message\": \"Not found!\"\n}\n ```\n\n##### *Error*:\n| Name | Type | Description | Required | Pattern |\n|:-----|:----:|:------------|:--------:|--------:|\n| message | string |  | true |
